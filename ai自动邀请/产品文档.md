# 精选联盟达人自动邀约助手 — 产品文档

## 一、产品概览

| 项目 | 说明 |
|------|------|
| **插件名称** | 精选联盟达人自动邀约助手 |
| **版本** | v2.0.0 |
| **技术栈** | Manifest V3 + Vite + @crxjs/vite-plugin + Vanilla JS + Shadow DOM |
| **运行环境** | Chrome / Edge 浏览器（PC端） |
| **适配页面** | `buyin.jinritemai.com` — 抖音精选联盟商家后台 |
| **数据存储** | chrome.storage.local（本地加密存储，无第三方服务器） |

## 三、5 大核心面板功能

### 3.1 首页看板 (Dashboard)
- 今日邀约量/累计达人/今日成功 统计卡片
- 达人分层统计（头部/中腰部/素人/置换）
- 最近邀约记录时间线
- 实时数据刷新

### 3.2 达人筛选 (Filter)
- 关键词搜索（防抖，实时过滤）
- 高级筛选：粉丝区间、GMV 区间、口碑分、佣金率、类目、达人分层
- 纯佣金达人筛选开关
- 筛选标签显示 + 一键移除
- 达人卡片列表（头像/昵称/粉丝/GMV/评分/分层标签）
- 批量勾选/全选/反选
- 黑白名单批量操作

### 3.3 邀约任务 (Task)
- 任务配置：目标数量、发送间隔、话术分组
- 待邀约队列预览（含筛选和黑白名单过滤）
- 白名单优先排序
- 运行控制台：开始/暂停/停止
- 实时进度条 + 当前邀约达人
- 运行日志（成功/失败实时显示）

### 3.4 数据分析 (Analytics)
- 达人总数/累计邀约/邀约成功率
- 邀约成功/失败 柱状图
- 达人分层分布图
- 粉丝数量分布图
- 佣金率分布图
- 数据导出：达人列表 CSV / 邀约记录 CSV / 全部数据 JSON

### 3.5 系统设置 (Settings)
- 产品信息配置（名称/佣金/链接/类目/福利）
- 话术模板管理（CRUD + 分组 + 启用/停用）
- 7 个变量：{达人昵称} {商品名} {佣金比例} {产品链接} {类目} {店铺名} {福利政策}
- 黑白名单列表管理
- 默认邀约参数
- 数据备份/恢复/清空

### 达人分层算法
| 分层 | 条件 | 标签颜色 |
|------|------|----------|
| 🔶 头部 | 粉丝 ≥ 100万 或 GMV ≥ 1000万 | 黄色 |
| 🔵 中腰部 | 粉丝 10万-100万 或 GMV 100万-1000万 | 蓝色 |
| ⚪ 素人 | 粉丝 < 10万 | 灰色 |
| 🟢 置换 | acceptsExchange = true | 绿色 |

## 五、安装使用

### 5.1 构建
```bash
cd ai自动邀请
npm install
npm run build      # 构建到 dist/
```

### 5.2 加载插件
1. 打开 Chrome/Edge → `chrome://extensions/`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `ai自动邀请/dist/` 目录
5. 插件加载完成，工具栏出现图标

### 5.3 使用流程
1. 进入精选联盟商家后台 `buyin.jinritemai.com`
2. 浏览「达人广场」页面，插件自动拦截达人数据
3. 点击页面右侧浮动按钮 🔥 打开面板
4. 在「筛选达人」设置筛选条件
5. 在「邀约任务」配置并启动邀约
6. 在「数据分析」查看转化率、导出数据

## 七、技术说明

### 7.1 API 拦截机制
- 通过 `XMLHttpRequest.prototype` 和 `window.fetch` Monkey-patch
- 递归搜索 JSON 响应中 10 个已知字段名 (`data`, `list`, `records` 等)
- 达人识别：检查对象是否含 `fans_count`/`follower_count`/`nickname` 等字段
- URL 过滤：仅拦截含 `api`/`square`/`author`/`creator` 等关键词的请求

### 7.2 DOM 回退机制
- 每 3 秒扫描页面 DOM 卡片元素
- 提取昵称、粉丝数等可见数据
- 以昵称哈希作去重 ID
- API 数据优先级高于 DOM 数据

### 7.3 Shadow DOM 隔离
- 所有 UI 渲染在 Shadow DOM 内，样式完全隔离
- 不影响精选联盟后台页面原有样式
- FAB 按钮 fixed 定位，z-index: 2147483647

### 7.4 存储方案
- `chrome.storage.local` — 本地加密存储
- 数据结构：templates, product, settings, blacklist, whitelist, inviteRecords, stats, history
- `unlimitedStorage` 权限，无容量限制
