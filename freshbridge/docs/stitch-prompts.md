## 全局设定（生成任何页面前先贴在对话开头）

**品牌信息：**
- 产品名称：鲜桥 FreshBridge
- 定位：精致高端水果农产品代卖平台
- 目标用户：水果产地农户、档口批发商、采购商
- 品牌关键词：品质感、信任感、自然感、山姆会员店风格

**视觉规范（固定）：**
- 主色：`#0F3B2C`（深森林绿）导航栏、主按钮
- 金色：`#C9A96E`（香槟金）VIP标识、购买按钮、强调
- 浅金：`#F5ECD7`（象牙金）会员卡、精选推荐背景
- 背景：`#F5F5F5`（暖灰白）
- 卡片：`#FFFFFF` 白底，圆角 `12rpx`，阴影 `0 2rpx 16rpx rgba(0,0,0,0.06)`
- 主文字：`#1A1A1A`，副文字：`#6B7280`，弱文字：`#9CA3AF`
- 图标：SVG 线框风格（Lucide/Heroicons），禁用 emoji 做图标
- 字体：标题 Playfair Display / 正文 Inter / 中文回退 PingFang SC
- 按钮：高度 `88rpx`（44px），圆角 `12rpx`，触摸区域 ≥44px
- 过渡动画：150-300ms

**平台规范：** 符合微信小程序设计规范，页面宽度 375px，底部 Tab 导航栏（行情/货源/发布/消息/我的），顶部导航栏深绿底白字

## 1. 首页（行情中心）

设计一个水果农产品交易平台的首页。

功能模块从上到下：
1. **Banner 轮播区**：3 张轮播图，高度 320rpx，圆角 16rpx，水果实拍图背景，白色大字标题，金色指示器圆点
2. **4 列快捷入口网格**：品牌商城（购物袋图标）、全链路服务（时钟图标）、大宗供需（趋势图图标）、品种排行（排行榜图标），每项带浅色圆角底框 + 线框图标 + 灰色小字标签
3. **行情快讯滚动条**：浅金底色 `#F5ECD7`，左侧小图标，文字单行垂直滚动播报
4. **统计数据行**：三列（在售品种数/上涨数/下跌数），白底卡片，数字粗体大号，上涨绿色下跌红色
5. **热门品种排行卡片**：排名序号徽章（第1名金色、第2名银色、第3名铜色），变体名 + 市场名 + 价格 + 涨跌幅箭头，最多显示 10 条，右上角"更多 ›"链接
6. **涨跌榜**：左右两列（涨幅榜/跌幅榜），各显示 5 条品种名 + 百分比
7. **价格走势区**：品种下拉选择器 + Canvas 折线图 + 各市场价格详情

整体页面灰底 `#F5F5F5`，每个模块独立白色圆角卡片，卡片间距 24rpx，页面底部留空 120rpx 避免被 Tab 栏遮挡。

# 页面跳转流程

> 每组流程单独粘贴到 Stitch，生成带交互的完整页面

---

## 流程 A：行情 → 品种排行

首页点击"品种排行更多" → 跳转品种排行页 → 点击返回箭头回首页

---

## 流程 B：品牌商城 → 商品详情 → 购物车 → 结算 → 订单

品牌商城首页商品网格 → 点击商品卡片进入详情 → 点击"加入购物车"或"立即购买" → 购物车/结算页 → 提交订单 → 我的订单页

涉及页面：品牌商城(底部Tab栏可见) → 商品详情(隐藏Tab栏) → 购物车(隐藏Tab栏) → 确认订单(隐藏Tab栏) → 我的订单(隐藏Tab栏)

---

## 流程 C：货源 → 类目筛选 → 品种列表

货源页 → 点击9宫格类目 → 筛选该品类品种 → 点击"发布货源"按钮

---

## 流程 D：发布 → 货源表单/采购表单

发布页 → 顶部Tab切换"发布货源"/"发布采购" → 填写表单 → 点击"立即发布" → 提交成功返回

---

## 流程 E：消息 → 消息详情

消息中心 → 点击消息项 → 底部弹出详情面板 → 标记已读

---

## 流程 F：我的 → 子页面跳转

个人中心 → 点击菜单项跳转：
- 代卖管理 → 代卖交易列表
- 销售记录 → 按日期分组的记录
- 结算对账 → 月度结算汇总
- 运单大厅 → 物流运单列表
- 在途追踪 → 地图+物流时间线
- 我的订单 → 商城订单列表

---

## 流程 G：记账 → 扫码/手动

代卖管理页 → 点击"记一笔" → 快速记账页 → 扫码或手动输入 → 提交记账

---

## 流程 H：大宗供需 → 发布供需

大宗供需页 → 点击底部"发布货源"/"发布采购" → 跳转发布页对应Tab

---

## 流程 I：物流追踪

运单大厅 → 点击"接单" → 进入在途追踪 → 查看物流时间线 → 确认到达
