## 一、完整页面布局结构（从上至下）

### Zone 1 — 顶部品牌栏 + 消息入口

**位置：** 页面顶部固定（sticky，不随滚动消失）  
**高度：** 88rpx（状态栏 44rpx + 内容区 44rpx）  
**内容：**
- 左：品牌 Logo 图文组合（"鲜桥" 汉字 + 绿叶图形标识，深绿色 `#0F3B2C`）
- 右：消息通知图标（铃铛 SVG，带未读红色数字角标）

**交互：** 点击铃铛 → 跳转消息中心页 `/pages/message/index`  
**背景：** 白色 `#FFFFFF`，无阴影，与内容区融合

---

### Zone 2 — 顶部广告资讯栏

**位置：** 品牌栏下方，紧随其后  
**高度：** 80rpx，紧凑单行横条  
**布局：** 横向滑动容器（`scroll-view`），内含 3-5 条资讯条目  
**条目样式：**
- 红色重点标签（`#DC2626` 背景，白色文字，圆角4rpx） + 文案（最多18字截断）
- 条目间以竖线分隔（`#E5E7EB`）

**示例文案：**
| 标签 | 文案 |
|------|------|
| 行情 | 今日黄瓜产地价上调，西红柿持续走低 |
| 公告 | 平台实名认证功能已上线，请尽快认证 |
| 推荐 | 山东寿光优质西红柿基地直供，限时对接 |

**交互：** 任意条目点击 → 跳转资讯详情页 `/pages/info/detail?id=xxx`（新建页面）  
**背景：** 浅绿色 `#F0FDF4`

---

### Zone 3 — 核心功能入口区（两大主按钮）

**位置：** 资讯栏下方，间距 24rpx  
**布局：** 左右两列并排，各占 (750 - 48 - 16) / 2 = 343rpx 宽  
**高度：** 200rpx

**左按钮 — 发布供应：**
- 背景：浅绿色渐变 `linear-gradient(135deg, #E8F5E9, #C8E6C9)`
- 图标：SVG 线条风格 "上传/供给" 图标（深绿 `#0F3B2C`，64rpx）
- 主文字："发布供应"（36rpx，粗体，`#0F3B2C`）
- 副文字："上架货源，找买家"（24rpx，`#6B7280`）
- 右下角小标签："农户入口"（浅绿底绿字小圆角标签）
- 点击 → `/pages/publish/index?type=supply`

**右按钮 — 发布求购：**
- 背景：暖橙色渐变 `linear-gradient(135deg, #FFF7ED, #FFEDD5)`
- 图标：SVG 线条风格 "搜索/采购" 图标（橙色 `#D97706`，64rpx）
- 主文字："发布求购"（36rpx，粗体，`#D97706`）
- 副文字："发布需求，找货源"（24rpx，`#6B7280`）
- 右下角小标签："采购商入口"（橙底橙字小圆角标签）
- 点击 → `/pages/publish/index?type=buy`

---

### Zone 4 — 农产品行情溯源模块

**位置：** 双按钮下方，间距 32rpx  
**标题行：** 
- 左侧：深绿竖条指示器（6×22rpx）+ "行情溯源"（32rpx，粗体，`#1A1A1A`）
- 右侧："更多 ›"（26rpx，`#0F3B2C`，点击跳转品种排行完整页）

**品类列表：** 横向滑动卡片（`scroll-view`，每卡片宽度 280rpx，间距 16rpx）

**每张品类卡片（以"黄瓜"为例）：**

```
┌──────────────────────────┐
│  🥒  黄瓜               │  ← 品类名 30rpx 粗体
│  产地报价                │  ← 标签 22rpx 灰色
│  ¥2.13      △ 2.02      │  ← 实时价(36rpx 粗体绿) + 差价(24rpx 红/绿箭头)
│  ──────────────────────  │
│  上海辉展  ¥2.13  ▼2.4% │  ← 各市场价格行 24rpx
│  广州江南  ¥2.78  ▼0.4% │
│  北京新发地 ¥1.52 ▲1.1% │
└──────────────────────────┘
```

**品类列表（严格按手绘草稿）：**
1. 黄瓜 2. 西红柿 3. 土豆 4. 大米  
（后续扩展：辣椒、茄子、豆角、苹果 — 横向可滑动查看更多）

**交互：** 点击任意品类卡片 → 跳转该品类详细报价对比页 `/pages/price-detail/index?variety=黄瓜`（新建页面）

**差价颜色规范：**
- 涨价：红色字体 `#DC2626`，上箭头 `▲`
- 跌价：绿色字体 `#16A34A`，下箭头 `▼`
- 持平：灰色 `#9CA3AF`，横线 `—`

---

### Zone 5 — 价格对比工具快捷入口

**位置：** 行情卡片下方，间距 20rpx  
**布局：** 单行横条卡片，高度 100rpx  
**内容：**
- 左侧：图标（两个箭头交叉的 SVG）+ 文字 "价格对比工具"
- 副文字："多市场同品类一键比价"
- 右侧：深绿箭头按钮 "立即对比 ›"
- 背景：浅金色渐变 `linear-gradient(135deg, #F5ECD7, #FFF)`

**交互：** 点击 → `/pages/price-compare/index`（新建页面）

### Zone 6 — 配套增值服务板块

**位置：** 价格对比下方，间距 32rpx

**三列网格（3 项配套服务）：**

| 序号 | 图标 | 服务名 | 副标题 | 跳转类型 |

### Zone 7 — 平台公告/资讯弹窗入口

**位置：** 服务区下方，间距 32rpx

## 三、视觉配色、字体、卡片样式规范

### 3.1 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色-深绿 | `#0F3B2C` | 品牌标识、主按钮、标题指示器 |
| 辅助-浅绿 | `#E8F5E9` | 供应按钮背景、行情模块底色 |
| 辅助-暖橙 | `#FFF7ED` | 求购按钮背景、橙色高亮 |
| 点缀-红色 | `#DC2626` | 涨价箭头、重要公告标签 |
| 点缀-金色 | `#C9A96E` | 公告喇叭、VIP标识 |
| 背景基色 | `#F5F5F5` | 全局页面底色 |
| 卡片白色 | `#FFFFFF` | 所有内容卡片 |
| 主文字 | `#1A1A1A` | 标题、正文 |
| 副文字 | `#6B7280` | 辅助说明 |
| 弱文字 | `#9CA3AF` | 时间、标签 |

### 3.2 字体规范（750rpx 设计稿）

| 层级 | 字号 | 字重 | 颜色 | 用途 |
|------|------|------|------|------|
| H1 页面标题 | 40rpx | 700 | `#1A1A1A` | 模块主标题 |
| H2 模块标题 | 32rpx | 600 | `#1A1A1A` | 区段标题 |
| H3 卡片标题 | 30rpx | 600 | `#1A1A1A` | 品类名、服务名 |
| Body 正文 | 28rpx | 400 | `#6B7280` | 描述、副标题 |
| Price 大价格 | 36rpx | 700 | `#0F3B2C` | 实时价格 |
| Price 差价 | 24rpx | 600 | `#DC2626`/`#16A34A` | 涨跌差价 |
| Caption 标签 | 22-24rpx | 400 | `#9CA3AF` | 时间、来源 |

### 3.3 卡片样式

- 白底：`#FFFFFF`
- 圆角：12rpx（模块卡片）、16rpx（大按钮）
- 阴影：`0 2rpx 16rpx rgba(0,0,0,0.06)`
- 间距：模块间 32rpx，内容间 24rpx
- 页边距：24rpx

## 五、可直接交付 UI 设计师的标准化需求文案

### 5.1 项目信息

- **项目名称：** 鲜桥（FreshBridge）
- **产品定位：** 农产品产地供需对接小程序
- **核心用户：** 农户、产地供货商、收购批发商
- **平台：** 微信小程序，750px 标准适配
- **设计风格：** 清新田园农产品风 — 浅绿+米白+暖橙点缀，简约卡片式，接地气但不土气

### 5.2 页面列表（首页范围）

1. **首页（行情中心）** — 本次设计范围
2. 发布页（货源/求购双模式）— 已开发
3. 品种详情页 — 待新建
4. 价格对比页 — 待新建
5. 服务详情页 — 待新建
6. 资讯详情页 — 待新建

### 5.3 首页功能模块清单（从上至下）

| 序号 | 模块名称 | 关键要素 |
|------|----------|----------|
| 01 | 品牌栏 | Logo、铃铛通知图标 |
| 02 | 广告资讯栏 | 横向滚动、红标+文案、3-5条 |
| 03 | 核心功能入口 | 双按钮：发布供应(绿) + 发布求购(橙) |
| 04 | 行情溯源 | 4+品种横向滑动卡片、实时价格、差价箭头 |
| 05 | 价格对比工具 | 横条卡片，引导比价 |
| 06 | 配套增值服务 | 3列网格，物流/农资/结算 |
| 07 | 平台公告 | 公告卡片，点击弹详情 |
| 08 | 底部 TabBar | 5 个 Tab |

### 5.4 设计优先级

1. **P0（必须有）：** 品牌栏、双按钮、行情溯源卡片
2. **P1（重要）：** 广告资讯栏、配套服务、平台公告
3. **P2（优化）：** 价格对比入口、品牌细节动效
